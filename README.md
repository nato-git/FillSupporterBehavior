## FillSupporterAddon

### 環境

このアドオンはマインクラフト統合版で動かすことができます。

---

### 実行方法

- リリースにある FillSupporterAddon をダウンロードし、ダウンロードしたファイルをクリック
- ワールドを開くときに、ビヘイビアーパックから FillSupporterAddon を有効にする
- チャットで/scriptevent fill:open を入力(この時、チートの有効化をオンにしておく)

---
